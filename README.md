# PIXI SHOOTER 🚀

PixiJS と TypeScript で制作した、ブラウザで遊べる本格シューティングゲームです。

## 🎮 プレイはこちら
[ここにあなたの公開URLを貼る](https://hirof35.github.io/my-pixi-game/)

## ✨ 特徴
- **HPシステム**: 3回まで被弾OK！緊張感のあるバトル
- **ボス戦**: 雑魚を10体倒すと出現する強力なボス
- **パーティクル演出**: 敵撃破時の派手な爆発エフェクト
- **レスポンシブな操作**: キーボード（矢印キーで移動、スペースで射撃）

## 🛠 使用技術
- **PixiJS v8**: 高速な2Dレンダリング
- **GSAP**: 滑らかなアニメーションと演出
- **Vite**: 高速なビルドと開発環境
- **TypeScript**: 安全で堅牢なコード設計
- **GitHub Actions**: 自動デプロイ環境の構築

## 🚀 ローカルでの実行方法
```bash
npm install
npm run dev
